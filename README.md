## Project Overview
This project is a Travel Companion App developed in Android Studio. The purpose of the app is to help international travellers convert important values quickly and easily.

The app supports three main conversion categories:
- Currency conversion
- Fuel conversion
- Temperature conversion

## Features
- Two spinner dropdown menus for selecting source and destination units
- Input field for entering the value
- Convert button
- Result display text view

## Supported Conversions

### Currency
- USD to AUD
- USD to EUR
- USD to JPY
- USD to GBP
- and reverse conversions through USD

### Fuel and Distance
- mpg to km/L
- km/L to mpg
- gallon to liters
- liters to gallon
- nautical mile to kilometers
- kilometers to nautical mile

### Temperature
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius
- Fahrenheit to Kelvin
- Kelvin to Fahrenheit

## Conversion Rates and Formulas
- 1 USD = 1.55 AUD
- 1 USD = 0.92 EUR
- 1 USD = 148.50 JPY
- 1 USD = 0.78 GBP
- 1 mpg = 0.425 km/L
- 1 gallon = 3.785 liters
- 1 nautical mile = 1.852 kilometers
- Fahrenheit = (Celsius × 1.8) + 32
- Celsius = (Fahrenheit − 32) / 1.8
- Kelvin = Celsius + 273.15

## How to Run the App
1. Open the project in Android Studio.
2. Allow Gradle to sync.
3. Run the app on an Android emulator or physical device.
4. Select the units, enter a value, and press Convert.
